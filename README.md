# Real-Time System Log Monitor
This project is a Python-based tool designed to simulate and monitor system logs in real time. It provides a practical environment for developers and DevOps engineers to observe server health and test log processing logic.
## Key Features
 * **Real-Time Simulation:** Automatically generates active testing logs to emulate a live production environment.
 * **System Diagnostics:** Includes simulated boot sequences and hardware checks.
 * **Proactive Warnings:** Simulates low-level system alerts, such as disk space warnings, to test monitoring response times.
## Prerequisites
 * Python 3.x installed on your machine.
